    Data Logging to SD Card on Raspberry Pi Hardware

    You can use the Simulink® Support Package for Raspberry Pi™ Hardware to log data from Simulink blocks to an SD card mounted on the Raspberry Pi Hardware. The data values can be saved in three formats: Structure, Structure with time, or Array. The data is logged to a MAT-file.

    Currently, data logging is supported only for these blocks:

    • Output Port

    • Scope

    • To Workspace

    You must install the Simulink Support Package for Raspberry Pi Hardware to use this support. To install or update this support package, perform the steps described in Install Support for RaspberryPi Hardware.

    For more information, see Simulink Support Package for Raspberry Pi Hardware.

    Raspberry Pi Hardware: Support added for Audio File Read block

    You can use the Audio File Read block to read audio frames from an audio file into the Simulink model.

    You must install the Simulink Support Package for Raspberry Pi Hardware to use this support.

    Raspberry Pi Hardware: Support added for customized Raspbian operating system using new hardware setup workflow

    The MATLAB® and Simulink support packages for Raspberry Pi hardware now provide a new hardware setup workflow, using which, you can customize the Raspbian Linux operating system on your Raspberry Pi™ hardware. These Raspberry Pi support packages require a set of libraries and packages that must be installed on the hardware for them to work properly. They do not support the generic Raspbian Linux operating system on your hardware. With the new hardware setup workflow, you can choose to overwrite or customize the existing operating system for it to be compatible with these Raspberry Pi support packages.

    You must install the Simulink Support Package for Raspberry Pi Hardware to use this support. To install or update this support package, perform the steps described in Install Support for RaspberryPi Hardware.

    For more information, see Simulink Support Package for Raspberry Pi Hardware.
