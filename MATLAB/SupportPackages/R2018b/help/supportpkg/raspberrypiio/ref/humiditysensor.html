    Humidity Sensor

    Connection to Humidity Sensor

    expand all in page

    Description

    This object represents a connection from the MATLAB® software to a humidity sensor attached to the SenseHAT object on the Raspberry Pi™ hardware. To create this object use the mysensehat.HumiditySensor method.

    Creation

    Properties

    expand all

    OperationMode — Humidity sensor mode operation
    continuous (default) | one-shot | continuous

    The mode in which the humidity sensor operates. The sensor can read the humidity value once or read the humidity values continuously. When you create the sensehat object, you can specify this value.

    Example:

    Data Types: char

    OutputDataRate — Output data rate for the humidity sensor
    12.5 (default) | 1 | 7 | 12.5

    The rate at which the sensor outputs the data. When you create the sensehat object, you can specify this value.

    Example: 7

    Data Types: char
