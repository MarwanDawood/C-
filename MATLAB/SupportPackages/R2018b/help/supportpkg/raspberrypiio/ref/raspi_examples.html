    raspi_examples

    Open featured examples for this support package

    collapse all in page

    Syntax

    raspi_examples

    Description

    example

    raspi_examples opens the featured examples that are installed with the MATLAB® Support Package for Raspberry Pi™ Hardware.

    Examples

    collapse all

    Open Featured Examples

    In the Command Window, enter:

    raspi_examples

    A window displays the examples.
