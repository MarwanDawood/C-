    IMUSensor

    Connection to IMU Sensor

    expand all in page

    Description

    This object represents a connection from the MATLAB® software to an IMU sensor attached to the SenseHAT object on the Raspberry Pi™ hardware. To create this object, use the mysensehat.IMUSensor method.

    Creation

    Properties

    expand all

    GyroscopeOutputDataRate — Output data rate for the gyroscope
    119 (default) | 14.9 | 59.5 | 119 | 238 | 476 | 952

    The rate at which the gyroscope outputs the data. The rate is specified in hertz (Hz). When you create the sensehat object, you can specify this value.

    Example: 14.9

    Data Types: int8

    GyroscopeFullScaleRange — The full scale range for the output of the gyroscope
    245 (default) | 245 | 500 | 2000

    The maximum angular velocity the gyroscope is expected to measure. This is measured in degrees per second (dps). When you create the sensehat object, you can specify this value.

    Example: 245

    Data Types: int8

    GyroscopeBandwidthMode — Bandwidth mode selection for the gyroscope
    (default) | | | | | |

    The bandwidth mode selected for the gyroscope. When you create the sensehat object, you can specify this value.

    Example: 50

    Data Types: int8

    AccelerometerOutputDataRate(Hz) — Output data rate for the accelerometer
    119 (default) | 10 | 50 | 119 | 238 | 476 | 952

    The rate at which the accelerometer is expected to output the data. It is measured in hertz (Hz). When you create the sensehat object, you can specify this value.

    Example: 10

    AccelerometerFullScaleRange(g) — Full scale range for output of the accelerometer
    2 (default) | 2 | 4 | 8 | 16

    The full scale range for output of the accelerometer. It is measured in ‘g’ units. When you create the sensehat object, you can specify this value.

    Example: 2

    Data Types: int8

    AccelerometerBandwidth(Hz) — Bandwidth selection for accelerometer
    50 (default) | 50 | 105 | 211 | 408

    The bandwidth selection for the accelerometer. It is measured in hertz (Hz). It is measured in ‘g’ units. When you create the sensehat object, you can specify this value.

    Example: 105

    Data Types: int8

    MagnetometerOutputDataRate(Hz) — Output data rate for the magnetometer
    40 (default) | 0.625 | 1.25 | 2.5 | 5 | 10 | 20 | 40 | 80

    The output data rate for the magnetometer measured in hertz (Hz). When you create the sensehat object, you can specify this value.

    Example: 2.5

    Data Types: int8

    MagnetometerFullScaleRange(gauss) — Full scale range for output of the magnetometer
    4 (default) | 4 | 8 | 12 | 16

    The full scale range for the output of the magnetometer measured in ‘gauss’ units.

    Example: 8

    Data Types: int8
